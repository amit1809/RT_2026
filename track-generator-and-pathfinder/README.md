# Track Generator

A Python tool for generating tracks and finding optimal paths.

- Track generation tool is from https://scioly.org/forums/viewforum.php?f=454
- Path finder tool is created deepseek v3 with vscode + cline

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/frdeng/RT-2025.git
   cd RT-2025/track-generator
   ```

## Usage

The project contains two main scripts:

1. **pathfinder.py** - Contains pathfinding algorithms
2. **gen.py** - Contains track generation logic

Run the generator:
```bash
python gen.py
```

Run the pathfinder:
```bash
python pathfinder.py
```
